Icon example:

```js padded
  <Icon type="home" />
  <Icon type="setting" theme="filled" />
  <Icon type="smile" theme="outlined" />
  <Icon type="sync" spin />
  <Icon type="smile" rotate={180} />
  <Icon type="loading" />
  <br />
  <br />
  <div>
    Use tag to create an icon and set its type in the type prop. Specify the spin property to show spinning animation and the theme property to switch between different theme.
  </div>
```
